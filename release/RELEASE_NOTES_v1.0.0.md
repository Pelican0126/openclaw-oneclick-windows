# OpenClaw Installer v1.0.0

## 中文

### 亮点

- 发布版本升级到 `v1.0.0`
- 安装向导模型改为“预置选项优先 + 手写兜底”，降低空白等待
- 新增 Telegram Bot Token（安装向导）与配对码（维护中心）
- 模型目录加载与缓存策略优化，减少“模型不全”与卡顿感知
- 关闭窗口保留托盘后台运行，提升稳定性

### 下载建议

- 大陆用户优先下载：`OpenClawInstaller-v1.0.0-windows.zip`
- 解压后双击：`INSTALL_NOW.cmd`

## English

### Highlights

- Version bumped to `v1.0.0`
- Wizard model flow is now "preset options first + manual input fallback"
- Added Telegram Bot Token in wizard and pair code in maintenance center
- Improved model catalog loading/cache behavior for better completeness and responsiveness
- Closing window keeps tray/background behavior for better stability

### Download recommendation

- Mainland users: use `OpenClawInstaller-v1.0.0-windows.zip`
- After extract, run: `INSTALL_NOW.cmd`
