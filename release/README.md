# Release Folder (v1.0.0)

This folder is the publishing package for OpenClaw Installer `v1.0.0`.
本目录用于发布 `v1.0.0` 所有对外分发文件。

## Files to distribute / 发布文件

- `OpenClawInstaller-v1.0.0-windows.zip` (recommended for mainland users)
- `OpenClawInstaller-v1.0.0-setup.exe`
- `OpenClawInstaller-v1.0.0.msi`
- `SHA256SUMS.txt`

## Mainland distribution suggestion / 大陆分发建议

If users cannot access GitHub, upload `OpenClawInstaller-v1.0.0-windows.zip` to your domestic channel (for example 123Pan, enterprise drive, or OSS direct link).

## End-user install (CN/EN) / 用户安装

- Chinese: download ZIP -> extract -> double-click `INSTALL_NOW.cmd`
- English: download ZIP -> extract -> double-click `INSTALL_NOW.cmd`
